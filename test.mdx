# Przykładowa Funkcja TypeScript

Poniżej znajduje się przykładowa funkcja napisana w języku TypeScript.

```typescript
// Przykładowa funkcja TypeScript

/**
 * Dodaje dwie liczby i zwraca ich sumę.
 * @param {number} a - Pierwsza liczba.
 * @param {number} b - Druga liczba.
 * @returns {number} - Suma dwóch liczb.
 */
function addNumbers(a: number, b: number): number {
	return a + b;
}

// Przykładowe użycie funkcji
const result: number = addNumbers(3, 7);
console.log(`Wynik dodawania: ${result}`);
```
